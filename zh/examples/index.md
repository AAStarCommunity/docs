# 完整示例概览 (Examples Overview)

本章节包含 AAStar SDK 在各种角色和使用场景下的完整端到端示例。

## 可用示例

- **[运营商流程 (Operator Flow)](./operator-flow)**: 质押与管理 SuperPaymaster。
- **[社区流程 (Community Flow)](./community-flow)**: 注册社区并设置声誉规则。
- **[终端用户流程 (End User Flow)](./enduser-flow)**: 使用社区信用发送 Gasless 交易。
- **[多链配置 (Multi-Chain Setup)](./multi-chain)**: 在不同网络间配置和使用 SDK。

## 本地运行示例

大多数示例可以在本地 Anvil 实例上运行。请确保你已预先在本地部署了 [AAStar 合约](https://github.com/AAStarCommunity/SuperPaymaster)。

```bash
pnpm run example:local
```
